# 🎮 Unreal Engine 미니맵 & 시스템 구현 프로젝트

## 📝 프로젝트 개요

> 언리얼 엔진 5를 기반으로 미니맵 시스템, 블루프린트 로직, 레벨 디자인, 시네마틱 연출까지
> 
> 
> 게임 개발 핵심 기능을 직접 설계하고 구현한 **3인 팀 프로젝트**입니다.
> 
> 저는 **레벨 디자인, 미니맵 시스템, C++ 기반 Actor 설계, 블루프린트 로직**을 담당했습니다.
> 

---

## 👥 팀 구성

- **총원:** 3명
- **내 담당 (홍성윤):**
    - 전체 맵 레이아웃 및 레벨 디자인
    - 미니맵 시스템 설계 및 구현 (Material, Render Target, UI 연동)
    - `ABase_MiniMapActor` C++ 클래스 작성
    - 블루프린트 로직 설계 및 최적화
    - 커브 시스템을 활용한 카메라/맵 줌 효과
    - 보스 전투 시네마틱 연출 일부 제작
- **A 팀원:** 캐릭터 애니메이션 제작, 보스 AI 시스템 개발
- **B 팀원:** UI 구조 설계 및 사용자 경험(UX) 플로우 제작, HUD 디자인

---

## 🎯 개발 목표

- UI-게임플레이가 실시간으로 연동되는 **반응형 시스템** 구현
- 팀 협업을 통한 **시스템·디자인 파이프라인 경험** 축적

- 탐색한 구역만 시각화하는 **실시간 맵 시스템 구현**

---

### 🔹 미니맵 커브 시스템
